In order to use this node you have to connect servo
- for choosing ball on pin 12
- servo for shooting it on pin 14

default position for choosing is 65
it can go to 30 to choose left/right ball 30 is left 100 is right
and to 100 to choose the other one

default position for shooting is 180 
it has to go to 130 to shoot

both servos, after small delay, go back to initial position

topic for shooting servo is /shootBall

topic for choosing servo is /chooseBall
